# User Storefront

This is the customer-facing storefront for browsing products, placing orders, and completing payments.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Pinia

## What This App Does

- Product listing and product detail pages
- Checkout flow for member and guest users
- Payment result and order query pages
- User account center and order history

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
